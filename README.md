# 🎸 NANA’S 20TH BIRTHDAY — Interactive Event Invite

Este é um projeto de **Front-end Criativo** desenvolvido para um evento temático baseado no anime **NANA**. O objetivo foi criar uma experiência imersiva que reflete a dualidade das protagonistas (Nana Osaki e Nana "Hachi" Komatsu).

## Tecnologias Utilizadas

* **HTML5** (Estrutura semântica)
* **CSS3** (Flexbox, Grid, Animações complexas e Keyframes)
* **JavaScript Puro** (Manipulação de DOM, Audio API e Modais)

## Conceito de Design

O projeto utiliza uma interface de "divisão de tela" (Split Screen) para representar os dois universos da obra:

- **Lado Punk (Black Stones):** Estética grunge, paleta em preto/vermelho e tipografia inspirada na Vivienne Westwood.
- **Lado Sweet (Trapnest/Hachi):** Estética coquette, paleta em tons de rosa/creme e elementos florais.

## Funcionalidades Principais

- **Mecânica de Escolha:** O usuário é incentivado a escolher sua "vibe" para o dress code do evento.
- **Live Sound:** Integração de trilha sonora temática com controles customizados.
- **VIP Pass Modal:** Acesso ao convite detalhado via modal interativo.
- **Responsividade:** Interface adaptada para dispositivos móveis, simulando um ingresso digital de show.
